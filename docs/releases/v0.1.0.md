# v0.1.0 - Initial Laravel AI Skills Catalog

Initial public release of Syarif Laravel AI Skills: a Laravel-focused skill catalog for Codex, Claude Code, and other AI coding assistants.

## Highlights

68 installable Laravel skills for day-to-day framework work, covering architecture, controllers, validation, Eloquent, transactions, queues, security, Livewire, responsive UI testing, performance, AI workflows, and project planning.

## Added

- `laravel-app-skills` bundle with 68 installable skills.
- Core Laravel standards for architecture, controller cleanup, Form Requests, Actions/Services, Eloquent, transactions, queues, Livewire, security, and testing.
- Laravel 11/12 app guideline skill for version-aware development workflow.
- Laravel specialist skill for routing broad Laravel tasks to focused skills.
- Livewire development skill for component state, validation, actions, events, pagination, and tests.
- Laravel database optimization skill for N+1 prevention, indexes, caching, pagination, locks, and migrations.
- Responsive UI testing skill for Playwright audits across mobile, tablet, laptop, desktop, Livewire states, overflow, clipping, forms, tables, modals, navigation, dark mode, and visual regression.
- `skills.sh.json` grouping for a cleaner skills.sh repository page.
- `plugin-groups.json` for grouping all skills under one installable bundle.
- Root Codex plugin artifact metadata in `.codex-plugin/plugin.json` for codex-marketplace.com submissions.
- Codex/ChatGPT plugin marketplace metadata in `.agents/plugins/marketplace.json`.
- Claude plugin marketplace metadata in `.claude-plugin/marketplace.json`.
- Claude plugin package manifest under `plugins/laravel-app-skills/.claude-plugin/plugin.json`.
- Codex plugin package under `plugins/laravel-app-skills`.
- Root `AGENTS.md` and `CLAUDE.md` guidance following the Agent Skills repo pattern.
- Universal assistant manifest at `agent-skills.json` for generic AI agents and integration tooling.
- Universal usage guide at `docs/UNIVERSAL_USAGE.md`, plus GitHub Copilot instructions in `.github/copilot-instructions.md`.
- README badges for downloads, skill count, skills.sh installs, clone URL, and pull requests.
- Validation scripts for skill metadata, plugin grouping, generated packages, and skills.sh grouping.

## Installation

Install all skills:

```bash
npx skills add soden46/syarif-laravel-ai-skills -s "*" -y
```

Interactive install prompt:

```text
Select the Laravel App Skills group to select every skill, then press Enter.
```

Install globally for Codex:

```bash
npx skills add soden46/syarif-laravel-ai-skills -g -a codex -s "*" -y
```

Install as a Codex plugin:

```bash
codex plugin marketplace add soden46/syarif-laravel-ai-skills --ref main
codex plugin add syarif-laravel-ai-skills@syarif-laravel-ai-skills
```

Test as a local Claude Code plugin:

```bash
claude --plugin-dir ./plugins/laravel-app-skills
```

Install one skill:

```bash
npx skills add soden46/syarif-laravel-ai-skills --skill form-requests
```

List available skills:

```bash
npx skills add soden46/syarif-laravel-ai-skills --list
```

Use with a generic AI assistant:

```text
Read AGENTS.md, then use skills/using-laravel-standards/SKILL.md as the entry skill for this Laravel repo. Load focused skills from skills/<skill-name>/SKILL.md only when relevant to the task.
```

## Notes

- Public skill content is written locally and mapped to Laravel topics without copying third-party skill body text.
- `.agents/` is ignored because it is a local install target and should not be committed.
- Root `.codex-plugin/plugin.json` points directly to canonical `skills/` for codex-marketplace.com artifact scans.
- `plugins/laravel-app-skills` is generated local output for plugin packaging; generated skill copies are ignored in Git to keep submitted artifacts under the 128-file scan limit.
- `agent-skills.json` is generated output for universal assistant metadata.
- skills.sh and skills.laravel.cloud indexing can lag until skills.sh telemetry sees the public repository and Laravel Skills completes its import and security audits.
